# Simple_Calculator
A basic calculator built using Python that performs arithmetic operations such as addition, subtraction, multiplication, and division.

Features ->
✔ Addition (+)
✔ Subtraction (-)
✔ Multiplication (×)
✔ Division (÷)
✔ User-friendly interface
